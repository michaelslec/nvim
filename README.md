# Neovim

Ok, so I've wanted a really cool Neovim config for a long time. I've even gone as far as thinking I would create a  GUI for it written in either Go or Rust. For now, let's build an awesome configuration for Neovim.

# Goals
- Easily extensible
- Well-organized
- Suckless
- TDD-inspired
- Lean & Fast as hell
- Awesome completions
- Beautiful

Okay, that's a lot of things. We're going to start with, piece by piece, adding functionality that I need in my editor. After it works right, we'll make it look nice. Finally, this cycle will happen one feature at a time, only adding features I need as I'm working my job.
